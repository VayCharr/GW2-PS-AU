# Guild Wars 2 PowerShell Addon Updater

A simple Guild Wars 2 PowerShell addon updater

## Using

Place the files in a folder inside your GW2 root folder and run

File structure should be as follows:

Guild Wars 2/
├── ...
├── Gw2-64.exe
├── ...
└── GW2-PS-AU/
    ├── README.md
    ├── update_addons.bat
    └── update_addons.ps1
